# KAI Solutions — web stranica

Poslovna web stranica tvrtke KAI Solutions (custom softver i WMS rješenja za mala
i srednja poduzeća). Statični HTML/CSS/JS, bez frameworka i build koraka.

**Live:** https://shywera.github.io/web

## Struktura
- 5 stranica: početna, usluge, projekti, o nama, kontakt (+ 404)
- Responzivan dizajn, tamna tema, bez vanjskih ovisnosti
- Automatski deploy na **GitHub Pages** kroz GitHub Actions (`.github/workflows/deploy.yml`)

## Lokalni pregled
Otvorite `index.html` u pregledniku — nema build koraka.
